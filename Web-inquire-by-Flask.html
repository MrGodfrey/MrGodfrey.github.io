Ubuntu 操作

创建虚拟环境

sudo apt-get install python3.7 来安装python 3.7

sudo apt-get install python3-pip 来安装pip

在~/.bash_aliases 文件内加入

并运行source ~/.bash_aliases来将python 连接到python3.

sudo apt-get install python3-venv 来安装venv包

python -m venv env 来创建名为env的虚拟环境.

问题 创建目录的时候别乱用sudo,可用 ls -l来查看所有者权限

启动与关闭虚拟环境

source env/bin/activate 来启动虚拟环境

deactivate来关闭虚拟环境.
