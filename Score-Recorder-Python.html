Score-Recorder-Python

2019-04-04 | 2019-04-09 | 计算机

第一个Python程序，实现成绩记录，包括简单的GUI.

项目地址 Github

目标

  1. 实现从Excel文件中读取数据，并本地化保存
  2. 可以从本地保存的数据文件中恢复Excel文件
  3. 一个简单的GUI，可以只用键盘快速读入数据并修改
  4. 有一定的稳定性

实现记录

创建虚拟环境

使用 venv包 创建虚拟环境，可以解决各个包之间互相依赖的问题。

初始化

使用命令 python -m venv .\environment 初始化一个新的虚拟环境.

启动虚拟环境

问题 PowerShell 不允许没有签名的脚本运行，可以用Set-ExecutionPolicy -Scope CurrentUser RemoteSigned 允许脚本运行

用命令 environment\Scripts\Activate.ps1 启动虚拟环境

用命令deactive 关闭虚拟环境.

使用pip安装包

pip install 可以安装需要的包

python -m pip install --upgrade pip 可以更新pip

读取文件

读取Excel文件可以用 openpyxl

安装 使用命令 pip install openpyxl

教程 官方教程

注意 openpyxl的索引是从1开始的。

数据持久化

使用 Shelve 模块进行数据持久化.

数据条目类

首先先创建个人的条目类 PersonRecord

实现的方法包括:

  1. 初始化
  2. 设置分数
  3. 获取字符串显示方法
  4. ___str___ 方法格式化数据输出

数据库类

创建数据库类，封装底层数据持久化的实现细节。

注意从Excel中读入文件，以及写入到Excel都由该类提供接口实现。

界面

使用tkinter , python最简单的GUI模块，却非常强大。

Entry 部件用于实现输入。用textvariable 的设置来与其他部件进行通讯。

Text 部件用来处理输入以及输出结果.

最终实现的效果如同命令行一般，非常优秀。
