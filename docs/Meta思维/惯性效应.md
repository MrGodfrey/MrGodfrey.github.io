# 惯性效应


人的身体和思维, 如同质量较大的飞轮一般, 都会缓慢的改变状态. 

为了长久的达成某个目标, 例如身体健康或者博士毕业等等, 可以充分利用惯性, 将任务或者目标分配到每一天. 保证不管自己当天的状态如何波动, 长期来看都会有稳定的产出.

日常的小习惯， 如定时起床、一日三餐、定时与朋友保持联系，可以在不确定性极大的生活中，给自己控制感。从而作为锚来稳定自己的情绪。
