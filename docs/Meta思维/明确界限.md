# 明确界限

一开始就确立自己的界限，可以让别人更加尊重你。

意识到自己与他人可以有不同之处，不要用自己的价值观和道德观去要求他人。

关于自己的情绪，也要意识到人应当只为自己的情绪负责，识别到情绪的感染力。

## 小习惯

1. 做不了的事早点拒绝
2. 别人有负面情绪时注意开导自己
