# 市场比价

如何面对失业或者结构性的职业消失，一个好的办法是隔段时间把自己放在市场上看看，自己值多少钱。

人需要不断找自己喜欢的工作，不要相信当前的稳定。

> 愿意放弃自由来换取保障的人，他最终既得不到自由，也得不到保障 —— 哈耶克

## 小习惯

1. 关注市场和跳槽信息
